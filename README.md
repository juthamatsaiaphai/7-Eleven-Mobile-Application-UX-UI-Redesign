# 7-Eleven Mobile Application UX/UI Redesign

## Overview

This project focuses on redesigning the user experience (UX) and user interface (UI) of the 7-Eleven mobile application to improve usability, simplify navigation, and enhance the overall user experience. The redesign process was based on user research and user-centered design principles.

---

## Technologies

- FlutterFlow
- Figma
- UX Research
- UI/UX Design

---

## My Responsibilities

- Conducted user research to identify usability issues and user needs.
- Analyzed the existing application to improve user experience.
- Designed interactive UI/UX prototypes using Figma and FlutterFlow.
- Simplified application navigation and reduced the checkout process from five steps to three.
- Applied user-centered design principles to improve usability and accessibility.

---

## Project Workflow

1. User Research
2. Persona Development
3. User Journey Mapping
4. Wireframe Design
5. UI/UX Prototype Design
6. Usability Evaluation
7. Prototype Improvement
